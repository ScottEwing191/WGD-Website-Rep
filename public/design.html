Game Overview

In Doodle Course the objective is to get the ball from the start of the level to the end. The problem is that there is nopath for the ball to roll on. The player must build thepath themselves by placing platforms and other itemsaround the level to create a path that the ball can useto get to the end of the level.

Features

Moving Platform

This object has an initial position and a target position as well as a move time. When the player enters Play Mode this platform will continuously move back and forth between the initial position and target position. The time it takes to do this will be determined by the move time.

Jump Pad

The jump pad is an object that will add an immediate force to the ball in the direction that the jump pad is facing. This will happen when the ball collides with the jump pad.

Boost Zone

While the ball is inside the boost zone a continuous force will be added to the ball in the direction the boost zone is facing. Since all movable objects can be rotated, the boost zone is a good way for the player to get the ball up to a higher position

Spikes

Spikes will immediately destroy the ball as soon as they collide with it.

Visual Style

Visually the game looks like it has been drawn on a sheet of paper. We wanted to create a style where once a level had been completed it would seem like the player had sketched out the solution to that levels problem.

Influences

There are two main games which have influenced the style and gameplay of our game. The first is Doodle Jump and the second is Ultimate Chicken Horse.

Doodle Jump

Doodle Jump is a game which looks like it takes place on a sheet of paper. The characters and platforms in the game have a simple hand drawn style which is we wanted to replicate although we did this by modifying existing assets to make them look hand drawn rather than drawing them ourselves.

Ultimate Chicken Horse

Ultimate Chicken Horse Is a four player 2D platform game where the player compete to get from the start of the level from the finish. The Obstacles in the level are places by the players as the game progresses.
